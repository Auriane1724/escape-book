# EscapeBook

Application web de réservation d’escape game développée dans le cadre du
Titre Professionnel Développeur Web et Web Mobile (DWWM).

## Fonctionnalités

- Inscription et connexion utilisateur (JWT)
- Gestion des rôles (USER / ADMIN)
- CRUD des salles (ADMIN)
- Création de créneaux horaires
- Réservation de créneaux
- Annulation de réservation
- Protection contre les doubles réservations

## Technologies utilisées

- Node.js
- Express
- MySQL
- JWT
- HTML / CSS / JavaScript

## Installation

```bash
npm install
```
